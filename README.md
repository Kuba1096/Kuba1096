## 🛠️ Czym się zajmuję?

### 💻 Programowanie i WebDev
* **Aplikacje czasu rzeczywistego:** Twórca systemu *Interactive Smart Escape Room Controller* (v2.6) opartego na Node.js i Socket.io.
* **Pluginy Minecraft:** Tworzenie dedykowanych rozwiązań (komendy, eventy, integracje z bazami danych MySQL/SQLite, dynamiczne menu GUI).

### 💬 Systemy Społecznościowe (Discord)
* Kompleksowe tworzenie i automatyzacja serwerów Discord.
* Wdrażanie systemów weryfikacji, zaawansowanej moderacji, auto-ról oraz systemów zgłoszeń (Ticket System).
